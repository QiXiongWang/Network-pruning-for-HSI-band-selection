# Network-pruning-for-HSI-band-selection
===
Codes for 2020 IGARSS NEURAL NETWORK PRUNING FOR HYPERSPECTRAL IMAGE BAND SELECTION 

## Requirement
* Python >= 3.6
  * `python3 --version`
* PyTorch >= 1.0
  * `pip install torch torchvision`
